a) Using conditional independence, we can show that

     P(A = a, B = b, C = c)  =  P(A = a, B = b) P(B = b, C = c) / P(B = b).

     Thus, p  =  p12 p23 / p2

 

b) Using inclusion-exclusion,

     P((A = a, B = b) U (C = c))  =  P(A = a, B = b) + P(C = c) - P(A = a, B = b, C = c)

     Thus, p  =  p12 + p3 – (p12 p23 / p2)
